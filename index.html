About

Resiste is an acronym that stands for:

REally SImple Syndication To Email (RESISTE)

Resiste Console

ferris-resiste-console is a 100% pure Java stand-alone and self-contained RSS-to-email application (it even comes with its own Java installation).

Only 3 simple steps to get up and running:

  1. Configure your email account
  2. Configure the RSS/ATOM feeds
  3. Schedule the application to run as often as you want
